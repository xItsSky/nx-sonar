# Contributor Code of Conduct

As contributors and maintainers of `@itssky/nx-sonar`, we pledge to respect everyone who contributes by filing issues, updating documentation, submitting pull requests, reviewing code, providing feedback in comments, and otherwise participating in the project.

Communication through any of the project's channels — GitHub issues and discussions, pull request reviews, email — must be constructive and never devolve into personal attacks, trolling, public or private harassment, insults, or other unprofessional conduct.

We extend courtesy and respect to everyone involved in the project regardless of gender, gender identity, sexual orientation, disability, age, race, ethnicity, religion, or level of experience. We expect everyone contributing to this project to do the same.

If a member of the community violates this code of conduct, the maintainers may take action — removing comments, closing issues or pull requests, blocking accounts — as deemed appropriate.

If you are subject to or witness unacceptable behavior, or have any other concerns, please email **emo.quentin@gmail.com**. Reports will be handled in confidence.
